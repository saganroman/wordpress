== Description ==

MotoPress Content Editor - is a frontend WordPress Page Builder which enables to build websites visually. It's a complete solution for creating responsive pages without coding and simply by dragging and dropping content elements. MotoPress plugin is easily extendable, fully responsive and works with all WordPress themes. Unlike other builders it is not locked with shortcodes and doesn't leave a mess on your website after deactivation.

== Installation ==

1. Upload the MotoPress plugin to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
You'll find 'MotoPress' in your Wordpress admin panel.

Read the [usage documentation](https://motopress.com/documentation/wordpress/plugins/) on the MotoPress website
